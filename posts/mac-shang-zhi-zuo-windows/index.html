在 mac 上制作 windows10 启动盘，亲测可用

2022-05-17
2分钟阅读时长

来源：https://chinese.freecodecamp.org/news/how-make-a-windows-10-usb-using-your-mac-build-a-bootable-iso-from-your-macs-terminal/

准备

  1. windows iso 文件
  2. 一个大小至少 8G 的 U 盘
  3. mac 上安装好 homebrew

步骤 1

查看 U 盘的名称：

$ diskutil list

该命令会列出电脑上所有的磁盘设备，可以通过大小来确认哪个是你插入的 U 盘，比如我的是：

/dev/disk8 (external, physical):
   #:                       TYPE NAME                    SIZE       IDENTIFIER
   0:     FDisk_partition_scheme                        *31.0 GB    disk8
   1:               Windows_NTFS WINDOWS10               31.0 GB    disk8s1

那么 disk8 就是我的 U 盘的名称了。

步骤 2

格式化 U 盘：

$ diskutil eraseDisk FAT32 "WINDOWS10" MBR disk8

这里的 disk8 要替换成你的 u 盘名称，“WINDOWS10” 是格式化后对硬盘的重命名，可以指定为任意值。

Started erase on disk8
Unmounting disk
Creating the partition map
Waiting for partitions to activate
Formatting disk8s1 as MS-DOS (FAT32) with name WINDOWS10
512 bytes per physical sector
/dev/rdisk8s1: 60507232 sectors in 1890851 FAT32 clusters (16384 bytes/cluster)
bps=512 spc=32 res=32 nft=2 mid=0xf8 spt=32 hds=255 hid=2048 drv=0x80 bsec=60536832 bspf=14773 rdcl=2 infs=1 bkbs=6
Mounting disk
Finished erase on disk8

提示以上信息说明格式化成功。

步骤 3

挂载 windows iso 文件：

$ hdiutil mount /Volumes/c/Win10_2004_Chinese\(Simplified\)_x64.iso
hdiutil: mount failed - 资源暂时不可用

mout 后面是你的 iso 所在路径

也可以双击 iso 文件来进行挂载，因为我之前已经双击挂载过了，所以这里提示“资源暂时不可用”

挂载完成后，finder 的位置会显示一个名为 CCCOMA_X64FRE_ZH-CN_DV9 的文件（也可能是 CCCOMA_X64FRE_EN-US_DV9 ，取决于 iso 文件），也可以在 /Volumes 目录下找到，比如我的是 /Volumes/CCCOMA_X64FRE_ZH-CN_DV9（可以通过 ls /Volumes 查看），这个路径后面会用到

步骤 4

复制文件到 u 盘，Windows 10 ISO 中的一个文件 install.wim 现在太大而无法复制到 FAT-32 格式的 USB 驱动器，需要单独复制它。

首先运行此命令以复制除该文件之外的所有内容：

$ rsync -vha --exclude=sources/install.wim /Volumes/CCCOMA_X64FRE_ZH-CN_DV9/* /Volumes/WINDOWS10

（这里将 /Volumes/CCCOMA_X64FRE_ZH-CN_DV9/ 替换为你的目录，参考步骤 3，/Volumes/WINDOWS10 替换为你的 u 盘名，参考步骤 2）

然后继续创建你要将文件写入的目录：

$ mkdir /Volumes/WINDOWS10/sources

如果提示 File exists 说明该文件夹已存在，可以继续执行下面的步骤。

使用 Homebrew 使用此终端命令安装名为 wimlib 的工具，该工具用于分割 install.wim 为 2 个小于 4G 的文件，

$ brew install wimlib

然后运行这个命令：

$ wimlib-imagex split /Volumes/CCCOMA_X64FRE_EN-US_DV9/sources/install.wim /Volumes/WIN10/sources/install.swm 3800

同样的，需要将 /Volumes/CCCOMA_X64FRE_EN-US_DV9/ 这部分替换为你的路径，以及 /Volumes/WIN10/ 这部分。之后需要等待一会，直到提示以下信息：

Splitting WIM: 4193 MiB of 4193 MiB (100%) written, part 2 of 2
Finished splitting "/Volumes/CCCOMA_X64FRE_ZH-CN_DV9/sources/install.wim" # 制作完成

步骤 5

至此启动盘已经制作完成，弹出 u 盘，插入到要安装 windows 的电脑，并使用 u 盘启动便可以看到 windows 的安装界面了。

总结

在 mac 上制作 windows 启动盘还是有点小麻烦的，不像 windows 上直接使用软碟通那么简单方便，网上也有不少教程是过时错误的，比如使用 balenaEtcher ，这个软件自己都会弹出警告，无法刻录 windows，还有使用命令行的，要将 u 盘格式化为 MS-DOS 格式，因为 MS-DOS 格式无法一次写入 4G 以上的文件，会提示 File too large 错误，如果格式化为 ExFAT，虽然可以成功写入，但是 bios 无法识别出 U 盘。其实本文使用的方法也是命令行，只是对 4G 文件进行了特殊处理，总的来看这个方法简单有效，感谢这位大佬的教程。
