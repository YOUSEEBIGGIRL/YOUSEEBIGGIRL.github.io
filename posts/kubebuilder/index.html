kubebuilder 实践

2022-10-03
2分钟阅读时长

快速入门

下载 kubebuiler

$ os=$(go env GOOS)
$ arch=$(go env GOARCH)

$ curl -L -o kubebuilder https://go.kubebuilder.io/dl/latest/${os}/${arch}
$ chmod +x kubebuilder
$ sudo mv ./kubebuilder /usr/local/bin

创建一个项目

$ mkdir operator
$ cd operator
$ go mod init operator.example.com

# 我们将使用 my.domain 域，
# 所以所有的 API 组将是<group>.my.domain.
$ kubebuilder init --domain my.domain

创建一个 API

运行下面的命令，创建一个新的 API（组/版本）为 “webapp/v1”，并在上面创建新的 Kind(CRD) “Guestbook”。

$ kubebuilder create api --group webapp --version v1 --kind Guestbook

安装 kustomize

# cd 到当前项目目录的 bin 目录下
$ cd bin
# go version≥go1.17
$ GOBIN=$(pwd)/ GO111MODULE=on go install sigs.k8s.io/kustomize/kustomize/v4@latest
# 查看当前目录下是否下载好 kustomize 
$ ls
controller-gen kustomize

将 CRD 安装到集群中

这步操作需要你已经安装好 kustomize

$ make install

继续沿用 快速入门 里已经创建好的项目

注意这里的 group 必须要指定为和 快速入门 中相同的 webapp，如果指定为别的，会报错：Error: failed to create API: unable to inject the resource to "base.go.kubebuilder.io/v3": multiple groups are not allowed by default, to enable multi-group visit https://kubebuilder.io/migration/multi-group.html

$ kubebuilder create api --group webapp --version v1 --kind CronJob
